# TaskHub
Projeto Trainee do ICMC Júnior.

## Como Rodar o Projeto 
Siga as instruções abaixo para clonar o projeto e executá-lo em sua máquina local para fins de desenvolvimento e testes.

### Pré-requisitos
Certifique-se de ter instalado em sua máquina:
- Node.js
- npm ou yarn

### 1. Clonar o repositório
Abra o seu terminal e execute o seguinte comando para clonar o repositório:
```bash
git clone URL_DO_PROJETO
```

### 2. Rodar o projeto

#### Requisitos
- MySQL instalado e rodando
- Crie um arquivo .env na pasta backend com as credenciais do seu MySQL, veja o arquivo .env.example para um exemplo.

#### Frontend
Após a instalação das dependências:
- Entre na pasta frontend com o comando:
```bash
cd frontend
```
- Instale as dependências com o comando:
```bash
npm install
```
- Inicie o servidor de desenvolvimento com o seguinte comando:
```bash
npm run dev
```
O projeto será iniciado e geralmente estará acessível em ```http://localhost:PORTA``` (verifique no terminal qual porta foi atribuída).

#### Backend
Após a instalação das dependências:
- Entre na pasta backend com o comando:
```bash
cd backend
```
- Instale as dependências com o comando:
```bash
npm install
```
- Inicie o servidor de desenvolvimento com o seguinte comando:
```bash
npm run dev
```

#### Certifique-se de que tanto o servidor frontend e backend estejam rodando para o projeto funcionar corretamente.

### Backend
